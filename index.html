The “philosophy” of the internet

I think the idea of the internet is all about sharing. To achieve this, we connect the computers into a network. To communicate efficiently across the machines and people, we develop some rules for both data representation and transmission, such as the HTML format and the HTTP protocol.

Lesson 2 Creating a Structured Document

Why begin with web development?

First, it would be fun to know how to develop a webpage.
Second, the webpages we build can be easily visualized in realtime, which makes it more convenient to receive feedbacks or share the results with others.

The roles of HTML and CSS

HTML: the structure (or layout) of the webpage;
CSS: the style (or decoration) of the webpage;
Javascript: the interactive component of the webpage.

Tree-like structure and boxification

The joint effect of tree-like structure and boxification makes it more easy to divide a complicated webpage into several simple components. Also, such divisions facilitate the styling of the webpage in CSS.

Lesson 3: Adding CSS Style to HTML Structure

CSS

CSS stands for "Cascading Style Sheets." The main function of them is to control the style of HTML elements. Usually the minimum "unit" in HTML that CSS style apply to is a "class" structure. To add style to HTML structure, we may have to specify the directory of the CSS file within the "head" section of the HTML file.

Avoiding Repetition

Human are not good at repetition because it is both tedious and error-prone. That's why we invented computers, which are good at repetition. Therefore, to make the best use of computers, we should avoid manual repetition and let computers do their excellent jobs.
